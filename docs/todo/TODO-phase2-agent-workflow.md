## 5. 技术建议（与生图主链路解耦）
1. 新建独立模块或服务（`backend_text/`），避免污染生图 API。
2. 保留统一鉴权层，但拆分路由前缀（示例：`/v1/text/*`）。
3. 使用独立 Redis 前缀/实例保存会话与路由状态。
4. 工具调用层做 Provider 适配器，避免供应商锁定。
